# Juan Gaviria Marketplace

Aplicación web de **tienda virtual (Marketplace)** desarrollada como MVP utilizando **Angular y Spring Boot**.

---

## 🚀 Tecnologías

### Frontend
- Angular 21.1.4
- TypeScript
- HTML5
- CSS3

### Backend
- Java 21.0.8
- Spring Boot
- Spring Data JPA

### Base de Datos
- PostgreSQL 17.5

---

## 📦 Funcionalidades

- Gestion de usuarios
- Gestion de productos
- Flujo de carrito de compras
